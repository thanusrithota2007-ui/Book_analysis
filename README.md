# Book Market Exploratory Data Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on a dataset of bestselling books to
understand patterns in ratings, pricing, review counts, genres, and book length.

The analysis focuses on identifying statistical patterns and relationships that influence
book popularity.

---

## Dataset

The dataset contains information about bestselling books, including:

- Title
- Author
- Genre
- Price
- Rating
- Review Count
- Print Length
- Publication Year
- Book Format

---

## Data Preparation

Before analysis, the dataset was cleaned and prepared by:

- Handling missing values
- Converting columns to appropriate data types
- Preparing categorical variables for grouping analysis

---

## Exploratory Analysis

The following analyses were performed:

### Distribution Analysis

- Rating Distribution
- Price Distribution
- Review Count Distribution
- Print Length Distribution

### Categorical Analysis

- Genre Distribution
- Book Format Distribution

### Bivariate Analysis

- Rating vs Review Count
- Rating vs Price
- Genre vs Average Rating

### Trend Analysis

- Year-wise Price Trend
